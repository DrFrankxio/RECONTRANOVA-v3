3.- Ejecutar `>npm install express` en la ruta del fichero `server.js`.

4.- Ejecutar `>npm install socket.io` en la ruta del fichero `server.js`.

5.- Ejecutar comando `>node server.js` en la ruta del fichero `server.js`.
